# Задача на программирование: расстояние редактирования

Вычислите расстояние редактирования двух данных непустых строк длины не более <code>10<sup>2</sup></code>, содержащих строчные буквы латинского алфавита.

## Примеры

Sample Input 1:

```
ab
ab
```

Sample Output 1:

```
0
```

<hr/>

Sample Input 2:

```
short
ports
```

Sample Output 2:

```
3
```
